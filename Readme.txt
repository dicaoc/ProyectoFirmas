Instrucciones para compilar el programa:

1. Cambiar la extension del archivo ProyectoFirmas.txt a ProyectoFirmas.java en Kali Linux.
2. ejecutar desde una consola el comando javac -encoding ISO-8859-1 ProyectoFirmas.java donde descargó el archivo txt.
3. Usar el comando java Menu para ver el menu del programa.

Para ver un ejemplo de uso del programa, puede ver el video https://youtu.be/9HzwwEmLi70 montado en youtube.
